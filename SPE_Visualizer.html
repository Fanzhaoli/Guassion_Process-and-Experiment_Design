📖 使用说明与操作指南

背景：本工具连接 Python 后端，模拟自我匹配任务（Self-Matching Task）中，实验设计参数 (P,T,W) 对自我优势效应（SPE）的调节。

前置条件：启动 Python 后端服务器（否则仅显示本地 JS 预览）：

cd 1_Code\npython Python_for_Generate\Backend\server.py

操作步骤：

  • 1. 选模型：在下拉框中选择生成模型（Sigmoid+DDM / GP+DDM v2.4 / v2.4.5 / v2.5）
  • 2. 调参：拖动滑块调整 P/T/W，点击预设组或展开高级参数
  • 3. 运行：点击「运行模型生成数据」，后端自动运行模型并保存 CSV 到 2_Data/Generate_Data/
  • 4. 查看：结果面板显示 SPE、RT、正确率等统计摘要
  • 5. 下载：点击「下载 CSV」获取生成的数据文件
  • 6. 保存：点击「保存到历史」将结果存入本地历史供对比

模型版本说明：

  • Sigmoid+DDM：纯 S2 机制 + Euler DDM，最基础版本
  • GP+DDM v2.4：S2 + GP 混合参数 + Euler DDM
  • GP+DDM v2.4.5（推荐）：S2 + GP + NA/omission + Deadline DDM + 正a重采样
  • GP+DDM v2.5：基于6组真实实验条件的 GP + DDM

数据来源：Python 后端直接调用 S2 gen_data_jh.ipynb、Generate_Data_v2.4_runner.py、Generate_Data_v2.4.5.ipynb、Generate_Data_v2.5_runner.py 中提取的模型代码。
